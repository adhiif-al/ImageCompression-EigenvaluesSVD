Algoritma SVD untuk Kompresi Gambar

Website ini menggabungkan implementasi dari tiga kelompok. Semuanya menggunakan Singular Value Decomposition (SVD) — sebuah faktorisasi matriks yang memanfaatkan nilai eigen dan vektor eigen.

A = U · Σ · Vᵀ

A_k = U_k · Σ_k · V_kᵀ (k singular values retained)

Rasio data = k(m + n + 1) / (m × n)
01

Input Gambar & Konversi Matriks

Gambar dibaca dengan Pillow/OpenCV dan dikonversi menjadi array numpy. Tiap channel warna (R, G, B) diperlakukan sebagai matriks terpisah berukuran m × n.

02

QR Algorithm untuk Nilai Eigen

Kelompok 20004 menggunakan iterasi dekomposisi QR: A_{k+1} = R_k · Q_k sehingga matriks A konvergen ke matriks diagonal (nilai eigen). Matriks U = Q₁·Q₂·…·Q_k menyimpan vektor eigen dari A₀.

03

Hitung SVD (U, Σ, Vᵀ)

Dari nilai dan vektor eigen, matriks SVD dibentuk. Nilai singular diurutkan dari terbesar ke terkecil — nilai terbesar mewakili informasi paling penting dari gambar.

04

Truncated SVD Berdasarkan Persentase

User memasukkan tingkat kompresi (0–100%). Program menghitung jumlah k nilai singular yang dipertahankan. Semakin kecil k, semakin besar kompresi (namun kualitas menurun).

05

Rekonstruksi & Output

Gambar direkonstruksi: A_k = U_k · Σ_k · V_kᵀ. Nilai pixel di-clip ke rentang [0, 255], lalu disimpan sebagai gambar baru yang dapat diunduh dalam format PNG/JPEG/BMP/WEBP.
